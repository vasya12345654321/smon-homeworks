## Задание 1

### Использованные команды

```bash
sudo apt update
sudo apt install postgresql -y

wget https://repo.zabbix.com/zabbix/6.4/ubuntu/pool/main/z/zabbix-release/zabbix-release_latest_6.4+ubuntu24.04_all.deb
sudo dpkg -i zabbix-release_latest_6.4+ubuntu24.04_all.deb
sudo apt update

sudo apt install zabbix-server-pgsql zabbix-frontend-php php-pgsql zabbix-apache-conf zabbix-sql-scripts zabbix-agent -y

sudo -u postgres psql
CREATE DATABASE zabbix;
CREATE USER zabbix WITH PASSWORD 'zabbix';
ALTER DATABASE zabbix OWNER TO zabbix;
\q

zcat /usr/share/zabbix-sql-scripts/postgresql/server.sql.gz | sudo -u zabbix psql zabbix

sudo nano /etc/zabbix/zabbix_server.conf
DBPassword=zabbix

sudo systemctl restart zabbix-server zabbix-agent apache2
```

## Задание 2

### Использованные команды

```bash
sudo apt update
sudo apt install zabbix-agent -y

sudo nano /etc/zabbix/zabbix_agentd.conf

Server=10.0.0.11
ServerActive=10.0.0.11
Hostname=kali

sudo systemctl restart zabbix-agent

zabbix_agentd -t agent.ping
```
